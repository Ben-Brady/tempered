With Flask

1. Install Dependencies

pip install flask tempered

2. Setup Flask

from flask import Flask

app = Flask(__name__)

@app.get("/")
def get_index():
    return "<h1>Hello</h1>"

if __name__ == "__main__":
    app.run()

3. Setup Tempered

from flask import Flask
+ from tempered import Tempered

app = Flask(__name__)
+ tempered = Tempered(
    template_folder="./templates",
)

@app.get("/")
def get_index():
    return "<h1>Hello World</h1>"

if __name__ == "__main__":
    app.run()

4. Add a template

<!-- templates/index.html -->

from flask import Flask
from tempered import Tempered

app = Flask(__name__)
tempered = Tempered()

@app.get("/")
def get_index():
+   return tempered.render_template("index.html")

if __name__ == "__main__":
    app.run()

5. Using a layout

The problem right now is the response doesn't contain a properly formatted HTML document.

Instead of copying and pasting the correct formatting for every page, we can use a layout.

<!-- templates/layout.html -->
<!DOCTYPE html>
<html>
<head>
    <meta charset='utf-8'>
    <meta http-equiv='X-UA-Compatible' content='IE=edge'>
    <meta name='viewport' content='width=device-width, initial-scale=1'>
    <title>{% slot required title %}</title>
    <meta
        name="description"
        content="{% slot description %}My first tempered layout{% endslot %}"
    >
</head>
<body>
    {% slot %}
</body>
</html>

This layout contains threee slots: a title, description and default slot.

A slot allows a user of the layout to place contents there.

The empty slot "{% slot %}" refers to the content of the page, this is where any html is placed.

The title slot "{% slot required title %}" refers to the title of the page. Since it has the required attribute, it must be provided and does not have a default.

The description slot "{% slot description %}...{% endslot %}", refers to the description of the page. If not provided the default contained within the two tags is used. A default value is required is the slot has not been marked required.

<!-- templates/index.html -->
+ {% layout "layout.html" %}

{% block title %}Home{% endblock %}
{% block description %}This is my home page{% endblock %}

<h1>
    Hello World
</h1>
