Parameters are optional

```html
<script type="tempered/metadata">
    parameters:
        - foo: str
        - bar: str
</script>
```

## Default Values

If you want to specify a default value, you can do so

```html
<script type="tempered/metadata">
    parameters:
        - foo: str
        - bar: str
</script>
```

Note: This syntax isn't nice to work with, so may be changed in the near future
